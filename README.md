# Shilong Jin Academic Homepage

This repository hosts the personal academic homepage for Shilong Jin, built with the Academic Pages / Minimal Mistakes Jekyll template.

## Main Files

- `_config.yml`: site metadata, author sidebar, repository URL, contact links.
- `_pages/about.md`: homepage content and section layout.
- `_pages/publications.html`: publication listing.
- `_pages/cv.md`: Markdown CV.
- `_publications/`: structured publication entries used by the publications page and CV.
- `images/`: copied research, patent, internship, and hobby assets.
- `_sass/layout/_shilong.scss`: custom visual design layer.

## Local Preview

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

Then open `http://localhost:4000`.
